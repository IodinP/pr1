# Python Django приложение
-----
### Описание проекта
CampusHub-учебный портал на Django для публикации новостей, управления курсами и API для мобильногоклента, использует PostgreSQL, кеш Redis.

Сервис запускается локально в виртуалном окружении и может быть вынесен за обратный прокси.

### Пример кода для активации окружения

#### Создание окружения (универсально)
```python -m venv .venv```

#### Активация
#### Linux/macOS:
```source .venv/bin/activate```
#### Windows (CMD):
```.venv\Scripts\activate```
#### Windows (PowerShell):
```.venv\Scripts\Activate.ps```
